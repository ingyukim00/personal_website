Introduction to HTML

HTML, or Hypertext Markup Language, is the language used to describe the structure of content on the web.

History

HTML can trace its roots back to 1989 at CERN, the European Laboratory for Particle Physics.

As the popularity of the web increased in the early 90s, HTML underwent a series of rapid changes.

HTML timeline

  1. 1991: Tim Berners-Lee publishes HTML Tagswhich describes the initial 18 elements of HTML
  2. 1992: NCSA develops the Mosaic browser, which will eventually evolve into Netscape
